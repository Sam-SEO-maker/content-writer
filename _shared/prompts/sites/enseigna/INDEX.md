# Gutenberg Examples — Index

Blocs validés à charger sélectivement selon le contenu à générer.

| Fichier | Contenu |
|---|---|
| `base-blocks.html` | Paragraph (plain, strong, lien, centré, stat emoji), H2/H3 (plain, avec emoji) |
| `blockquote.html` | Citation mise en avant (accroche, promesse de marque) |
| `image.html` | Image full+caption (lien), large+caption, large sans caption, full redimensionnée |
| `list.html` | UL plain, UL avec strong (tarifs), OL numérotée avec emoji |
| `pros-cons.html` | Colonnes pros/cons — variante A titres `<p>`, variante B titres `<h3>` |
| `references.html` | H3 Références + OL d'URLs sources |
