# Enseigna — Champs ACF (template review « sites de formation »)

Champs ACF à renseigner dans WordPress pour chaque article d'avis Enseigna.
Ces champs **remplacent la fiche technique HTML** : les mêmes données vont dans ces champs ACF, pas dans un tableau dans le corps.

> Règles : note concurrent **max 4/5** (Superprof = référence). Pas de tiret cadratin. Données vérifiées en live (page officielle + Trustpilot + 1 source croisée).
